Glance

A discreet, ambient Claude assistant for your Mac menu bar. Hit a hotkey from any app and the answer floats in near your cursor.

⬇︎  Download for macOS View on GitHub
Free & open source · macOS 13+ · Apple Silicon & Intel · all downloads & .zip
⚠️ You need Claude Code. Glance has no API of its own — it drives your local claude command, signed in with your own Claude plan. If you don't have Claude Code installed and logged in (claude /login), Glance can't answer.

What it does

✶ Ask about a selection

Highlight text in any app and press ⌥⌘A. Glance reads it and answers, explains, translates, or rewrites it.

✶ Screenshot a question

Press ⌥⌘S, drag a box over anything — an error, a chart, text in an image — and ask about what's on screen.

✶ Private by default

The answer stays hidden until you hold ⌥. Nobody glancing at your screen can read it; let go and it vanishes.

✶ Your model, your plan

Choose Opus 4.8 or Sonnet 4.6, and Normal or Max effort — right from the menu bar. No API key, runs on your Claude subscription.

⌨️ Keyboard shortcuts

Two global hotkeys — they work in any app, no need to switch to Glance first.

ShortcutWhat happens
⌥⌘AGrab the selected text and answer about it.
⌥⌘SDrag a screen region (crosshair) and ask about it.
✨ menuAsk about Clipboard — answer about whatever you've copied.

In the answer panel:

KeyAction
Hold ⌥Reveal the answer (it's hidden until you do). Let go to hide.
↵Send a follow-up — keeps the conversation context.
⌘CCopy the whole answer (works even while hidden).
escClose. (Clicking back into your work also closes it.)

Install

Option A — download the app

  1. Download Glance.dmg, open it, and drag Glance into Applications.
  2. It's free and unsigned, so macOS Gatekeeper blocks it on first launch. Clear the quarantine flag once in Terminal: 
    xattr -dr com.apple.quarantine /Applications/Glance.app
    Then open it. Look for the ✨ in your menu bar.

Option B — build from source (no Gatekeeper step; read every line first):

git clone https://github.com/NatePearson/glance.git
cd glance
./build.sh
open Glance.app

First time you use ⌥⌘A, macOS asks for Accessibility permission (Glance sends a copy command to grab your selection). Screenshot mode needs no permission.
